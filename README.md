# skia-builder

| Platform | Supported          |
| -------- | ------------------ |
| Web      | :white_check_mark: |
| Android  | :white_check_mark: |
| iOS      |                    |
| Mac      | :white_check_mark: |
| Windows  |                    |
| Linux    |                    |


### Android
